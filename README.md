## Installation

Step-by-step instructions on how to set up and run the project locally.

### Clone the repository

```bash
git clone https://github.com/BagdalF/ecomm.git
```

### Navigate to the project directory

```bash
cd [(PROJECT_NAME)]
```

### Install dependencies

```bash
npm install
```

### Run the project

```bash
npm run dev
```
